== Description ==

Stick Post Widget plugin.Display recent stick post.

**Features**

1. Sticky recent post show.
2. Display post date with post like recent post widget
3. You will option to show only sticky post

More Detail : http://socialcms.wordpress.com/



== Installation ==


1. Upload the **stick-post-widget** folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go Appearance => widgets section  see Stick Recent Posts
